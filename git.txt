Git은 소프트웨어 형상 관리를 위한 도구

주요 기능 )
1. 버전 관리 
- 어떤 것이 변경 되었는지 추적 및 관리

2. 백업
- 로컬의 데이터는 언제 고장날지 알 수 없다. 
- 따라서 데이터 백업을 주기적으로 하는 것이 바람직 하다. 

3. 협업 
- git hub 같은 온라인에서 여러 사람이 함께 작업이 가능하다. 

git 사용법 ) 
1. 지정 프로젝트(혹은 폴더)를 git 저장소로 만들어야 한다. 
2. 변경점 추적을 위해서 사용자 이름/이메일 등록이 필요하다. 
3. 변경이 발생하면 버전을 생성해서 관리할 수 있다.
